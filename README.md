# VCP - Vector Context Protocol

Um protocolo aberto e agnóstico para recuperação de contexto semântico.

## 🚀 Instalação Rápida

```bash
uv add vcp[lancedb, langchain, valkey]
uv run vcp init
```

## 4 Operações Principais

1. **Ingest** - Inserir documentos
2. **Search** - Buscar documentos semelhantes
3. **Rerank** - Reordenar resultados
4. **Forget** - Remover documentos
